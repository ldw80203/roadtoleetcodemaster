- 能使用兩層 `for` 迴圈列舉不同索引的數字配對。

### Runtime 判讀

LeetCode 顯示的 runtime 百分比會受到伺服器負載、JVM 暖機與測量波動影響，不能只靠一次提交判斷能力。本解法真正需要改善的地方是演算法會隨輸入長度呈平方成長。

### 下一階段

1. 先讓程式找到答案時立刻結束，觀察實際執行次數如何下降。
2. 理解即使提前結束，最壞情況仍是 O(n²) 的原因。
3. 學習 `HashMap` 後，把「尋找另一個數」從逐一掃描改成接近一次查找，目標時間複雜度 O(n)。
