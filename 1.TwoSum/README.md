# 1. Two Sum

- 題目：[Two Sum](https://leetcode.com/problems/two-sum/description/)
- 語言：Java
- 狀態：通過
- 解法：雙層 `for` 迴圈列舉索引配對。
- 時間複雜度：`O(n²)`
- 空間複雜度：`O(1)`

## 學習紀錄

- `i` 使用外層迴圈，從 `1` 開始。
- `j` 使用內層迴圈，範圍為 `0` 到 `i - 1`。
- `j < i` 可避免重複檢查同一組索引，也避免使用同一元素兩次。
- 找到答案後使用 `return`，立即停止整個方法。
- `break` 只會停止目前所在的迴圈。
- 回傳陣列索引時，使用 `n[0] = j`、`n[1] = i`。
